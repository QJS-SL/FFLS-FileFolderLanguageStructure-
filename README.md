FFLS-FileFolderLanguageStructure

This is a language to help you with organizing your files in a structured manner.

//STRUCTURE

//Project, defines the top of the whole structure followed by {}

//Folder, defines a folder by name or structure

//File, definies a specific file

//ONE TIME

//Clean, removes all files from a certain folder without deleting the folder, once

//Remove, removes one specific file or folder, backups or child files get moved to the parent, once

//Delete, delete the corresponding file from the folder

//Add,adds one name with type and name in a folder, once

//Relocate, relocate certain files to a different folder

//Search, searches for a specific file, if it exist give back the file path and a boolean state

//FILTERS

//Exclude, all but one file type 

//Ignore, all but one specific file by name

//Include, all files but also drag along the ones that were original forbidden

//All, all files in a folder 

//RULES

//Forbid, prevends a certain file type to be added to a folder, running

//LOCATION

//To, points to a certain folder to be a added

//In, looks inside a certain folder
